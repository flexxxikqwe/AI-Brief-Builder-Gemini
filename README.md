# TaskFlow - Nuxt 3 Task Manager

A polished, full-stack task management application built as a hiring test assignment.

## Tech Stack

- **Frontend:** Nuxt 3, Vue 3, TypeScript, Pinia, Tailwind CSS
- **Backend:** Node.js, Express, JWT Auth
- **Icons:** Lucide Vue Next

## Features

- **Authentication:** Mock JWT login with persistent sessions
- **Task Management:** Create, Read, Update, Delete (CRUD) operations
- **Filtering & Search:** Real-time debounced search and status filtering
- **Sorting:** Sort by date, title, or priority
- **Responsive Design:** Fully mobile-responsive UI with a clean, modern aesthetic
- **State Management:** Centralized state using Pinia stores

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Start the backend API:
   ```bash
   npm run server
   ```
